# Instructions for using Personal Access Token with this repository
# 
# Option 1: Push with token in URL (temporary)
# git push https://YOUR_TOKEN@github.com/anilchaganti007/udaysite.git main
#
# Option 2: Store token in Windows Credential Manager
# When prompted during git push, use:
# Username: anilchaganti007
# Password: YOUR_PERSONAL_ACCESS_TOKEN
#
# Option 3: Use Git Credential Manager
# Run: git credential-manager-core configure
# Then push normally
